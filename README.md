# Retail-Sales-Analysis
Data Analyst Project using Python, MySQL, Excel,Power BI

# Project Overview
This project focuses on analyzing retail sales data to identify sales trends, top-performing products, and revenue insights.

# Tools & Technologies
- Python (Pandas, NumPy)
- Excel
- Power BI
- SQL (MySQL)

# Key Tasks Performed
- Data cleaning and preprocessing using Python
- Removed duplicates and handled missing values
- Performed Exploratory Data Analysis (EDA)
- Created sales KPIs and trend analysis
- Built interactive dashboards in Power BI

# Key Insights
- Identified top-selling products
- Monthly and yearly sales trends
- Region-wise performance analysis

# Project Output
- Cleaned dataset
- Python analysis notebook
- Power BI dashboard screenshots
